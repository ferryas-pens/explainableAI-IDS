Sistem deteksi intrusi berbasis machine learning sering menunjukkan performa sangat tinggi pada evaluasi within-dataset, namun reliabilitasnya menurun ketika diterapkan pada dataset dengan distribusi trafik dan karakteristik ekstraksi fitur yang berbeda. Penelitian ini menyajikan kerangka Explainable Artificial Intelligence (XAI) untuk Network Intrusion Detection System (NIDS) dengan pipeline bebas-kebocoran dan evaluasi generalisasi lintas-dataset. Eksperimen menggunakan CIC-IDS-2017 sebagai dataset referensi dan sepuluh partisi CSE-CIC-IDS2018 sebagai dataset target. Pipeline meliputi normalisasi label dan nama kolom, harmonisasi skema fitur berbasis referensi, stratified subsampling, pembagian latih-uji, seleksi fitur hanya pada data latih, standardisasi berbasis data latih, serta SMOTE hanya pada data latih. Model referensi yang tersimpan pada artefak eksperimen adalah Random Forest. Pada evaluasi within-dataset, model mencapai F1-score 0,9963, MCC 98,54%, dan AUC-ROC 0,9997. Namun pada evaluasi lintas-dataset, rata-rata F1 tertimbang masih tampak tinggi (0,7864) sementara MCC seluruh target menjadi 0,00%, dengan recall kelas serangan bernilai 0 pada seluruh partisi CSE-CIC-IDS2018. Hasil ini menunjukkan bahwa F1 tertimbang dan accuracy dapat menyesatkan pada dataset tidak seimbang ketika model cenderung memprediksi kelas mayoritas. Analisis SHAP mengidentifikasi Bwd Packet Length Min, Flow Duration, Fwd URG Flags, Bwd Packet Length Max, dan Average Packet Size sebagai fitur global terpenting, tetapi stabilitas fitur tidak cukup untuk menjamin generalisasi. Temuan ini menegaskan perlunya pelaporan MCC, recall per-kelas, dan validasi lintas-dataset dalam riset IDS.
Kata kunci: explainable AI, intrusion detection, generalisasi lintas-dataset, SHAP, MCC, data leakage
